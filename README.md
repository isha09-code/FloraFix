# 🚀 Problem Statement

Farmers often struggle with:
- Late disease detection  
- Lack of expert access  
- Overuse of chemicals  
- Low crop productivity  

---

# 💡 Our Solution

FloraFix provides:

✔ Instant AI-based disease detection  
✔ Ayurvedic eco-friendly treatments  
✔ Smart crop insights  
✔ Real-time monitoring
